O que vamos aprender?

Confira abaixo as tecnologias que você irá aprender neste cusro!

Ui & Ux Design

Aprenda a criar lindas interfaces, entender as heurísticas e transformar a ideia do seu cliente em arte com esse curso construido por especialistas na área de Ux Design e Ui Design!

Front-End

Transformar seus designs em páginas animadas e responsivas, aprenda de uma vez como construir páginas web com HTML, CSS, JavaScript de uma forma totalmente simples!

Back-End

Aprenda a criar desde funções simples até sistemas completos, tenha em seu arsenal as melhores práticas ao construir um sistema robusto e seguro! Aprenda PHP do básico à orientação a objeto construíndo projetos desde um simples gestor de usuários até um sistema empresarial(ERP)!

Mobile

Transforme suas ideias em Apps nativos para Android utilizando todo poder do Flutter! Aprenda desde como construir sua primeira tela até publicar sua aplicação na Google Play!

Frameworks

Chegou a hora de você dar próximo passo na programação! Aprenda a programar Front-End e Back-End de uma forma facilitada utilizando os melhores frameworks no mrcado como Vue.js e Bootstrap no Front-End e o poderoso Laravel no Back-End!

Extras

Domine tudo que é necessário para trabalhar em equipe, aprenda Git e Github, metodologias ágeis como Scrum, captação de clientes em plataformas de freelancers, hospedagem de suas aplicações web e até mesmo seu currículo e perfil no LinkedIn
